# String Operations GUI — Prefijos, Sufijos, Subcadenas y Cerraduras Σ* / Σ+

![Python](https://img.shields.io/badge/Python-3.x-blue)
![Tkinter](https://img.shields.io/badge/GUI-Tkinter-green)
![Status](https://img.shields.io/badge/status-active-success)

Aplicación con **interfaz gráfica en Python** para trabajar con operaciones fundamentales sobre **cadenas y alfabetos** utilizadas en **Teoría de la Computación y Lenguajes Formales**.

El programa permite calcular:

- Prefijos
- Sufijos
- Subcadenas
- Cerradura de Kleene (Σ*)
- Cerradura positiva (Σ+)

Todo mediante una **GUI sencilla desarrollada con Tkinter**.

---

# Vista general

La aplicación contiene **tres secciones principales**:

1. **Subcadenas / Prefijos / Sufijos**  
2. **Cerraduras Σ* y Σ+**  
3. **Ayuda e información**

Permite además **exportar los resultados a archivos `.txt`**.

---

# Características

✔ Interfaz gráfica fácil de usar  
✔ Cálculo de **prefijos, sufijos y subcadenas únicas**  
✔ Generación de **Σ*** y **Σ+** para un alfabeto  
✔ Soporte para diferentes formatos de alfabeto  
✔ Exportación de resultados  
✔ Protección contra generación excesiva de cadenas
